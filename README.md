On Par Preshift Agent

This project generates daily On Par Entertainment preshifts for an entire accounting period.

Sources:

* HR-Period Goals 2026
* HR-Memos
* HR-Core Value Shout Out (Responses)
* HR-Preshift Memory

Output:

* HR-Preshift

Features:

* Daily preshifts
* Weekly translations
* Daily questions
* Active memo tracking
* Core value shout outs
* Period-based goal alignment

Future:

* Google Docs integration
* Google Sheets integration
* Automated period generation
